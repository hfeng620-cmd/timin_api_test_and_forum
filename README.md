# 一个由社区共建的 AI 中转站观察站

我们用公开、可追踪、可修正的方式，持续整理 AI 中转站的价格、倍率、套餐口径、试用入口和真实体验，尽量减少群聊里的碎片信息。

- 看数据，不只看转述
- 结论可追踪，也能被修正
- 不会写代码，也能贡献线索和验证

如果你也长期在用各类中转站，欢迎一起来补数据、提线索、做验证，把它慢慢做成一个真正有用的公共项目。

## 现在这版已经有什么

- 默认 `白蓝` 工具站主题，保留 `白绿` 备用主题
- 首页改成更像“社区前台”的结构
- 榜单页不再硬套统一指标，而是按真实口径展示
- 已先录入一批真实中转站
- 首页和榜单页都留好了演示视频占位区

## 这个项目解决什么问题

很多中转站信息目前只散落在群聊里：

- 有的只有倍率，没有稳定性反馈
- 有的有注册送额，但没人整理入口
- 有的同站不同档位，容易被写成一个误导数字
- 有的“便宜”“稳定”只是口头口碑，没有统一口径

这个项目的目标不是做一个“像站一样的空壳”，而是做一个能被社区一起维护、能不断修正的观察站。

## 当前结构

- `/`
  首页，偏社区前台，负责速报、试用入口、讨论、待补实测和共建任务
- `/stations`
  核心入口页，负责按站点入口、套餐类型、倍率、价格口径和社区备注做比较

## 你现在就可以怎么参与

1. 发现线索
   新站点、价格变化、邀请码、试用入口、踩坑反馈
2. 补充证据
   原始链接、截图、价格口径、出现时间、模型分组
3. 提交入口
   `Discussion / Issue / PR`

不会写代码也没关系，先补一条准确信息就很有价值。

## 推荐协作方式

建议群里统一走这条路径：

1. 新线索先进 `Discussions` 或 `Issue`
2. 需要收录时开 `Issue`
3. 真正改站内容只认 `PR`

一句话版本：

`讨论先进 Discussions，确定要做再进 Issues，最终入站只认 PR。`
